Detailed Description

Output arguments wrapper.

Example: 

OutputArguments output(nlhs, plhs, 3);
MxArray cell = MxArray::Cell(1, 3);
cell.set(0, 0);
cell.set(1, 1);
cell.set(2, "value");
output.set(0, 1);
output.set(1, "foo");
output.set(2, cell.release());
